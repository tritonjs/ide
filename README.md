# ide

http-proxy powered cloud9 ide distributor.

## What does this do?

It uses `triton_username` and `triton_apikey` to find the docker container hosting
the user's c9 sdk instance.

## Install

```bash
git clone https://github.com/tritonjs/ide

# setup redis.

npm start
```
